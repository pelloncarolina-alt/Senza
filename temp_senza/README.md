# SENZA - Health & Recovery Website

## 📁 Estructura del Proyecto

```
senza/
├── index.html              # Página principal (HOME)
├── protocolos.html         # Página de protocolos de recovery
├── para-teams.html         # Página B2B para clubes y equipos
├── contacto.html           # Página de contacto y formulario
```

✅ **Scroll Suave** - Navegación fluida entre secciones
✅ **Colores SENZA** - Identidad visual correcta aplicada

## 🚀 Cómo Usar en VS Code

1. **Abre la carpeta en VS Code**:
   - Archivo → Abrir carpeta
   - Selecciona la carpeta `senza`

2. **Visualiza en el navegador**:
   - Opción A: Instala "Live Server" extension (click derecho en `index.html` → "Open with Live Server")
   - Opción B: Abre directamente `index.html` en tu navegador

3. **Edita los archivos**:
   - Todos los archivos están listos para editar
   - Los cambios se verán reflejados en vivo si usas Live Server

## 🖼️ Agregar Imágenes

1. Coloca tus imágenes en la carpeta `imagenes/`
2. En el HTML, referéncialas así:
   ```html
   <img src="imagenes/nombre-imagen.jpg" alt="Descripción">
   ```

3. Asegúrate de usar formatos optimizados:
   - JPG para fotografías
   - PNG para gráficos/iconos
   - WebP para mejor rendimiento (opcional)
